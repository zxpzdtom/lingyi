# 灵译隐私政策

最后更新：2026-07-02

灵译是一个本地优先的 Chrome 翻译扩展，用于提供划词翻译、整页翻译、双语对照和页面总结能力。

## 我们处理哪些数据

灵译只会在提供功能所需的范围内处理以下内容：

- 用户选中的网页文本。
- 用户主动触发整页翻译或页面总结时的当前页面文本。
- 用户在设置页保存的偏好设置，例如目标语言、界面语言、主题、模型、Prompt 模板和第三方 API 配置。

## 数据如何使用

这些数据仅用于完成用户请求的翻译、双语对照、页面总结和扩展设置同步到本机存储。灵译不会将这些数据用于广告、用户画像、信用评估、数据转售或与扩展核心功能无关的用途。

## 本地模型与第三方 API

默认的本地 Gemini Nano 模式会优先在浏览器本机处理文本，页面内容无需离开用户设备。

如果用户主动配置并启用 OpenAI、OpenRouter、Gemini 或其他 OpenAI-compatible API，灵译会把用户触发翻译或总结所需的文本发送到用户配置的服务端点。第三方服务对这些数据的处理受其各自隐私政策约束。

## 本地存储

灵译使用 `chrome.storage.local` 保存用户设置。第三方 API Key 或兼容端点配置如果由用户填写，也会保存在本机浏览器的扩展存储中。灵译不会把这些设置上传到我们的服务器。

## 权限说明

灵译请求网页访问权限，是为了在用户当前浏览的页面上显示划词翻译气泡、整页翻译结果和页面总结面板。扩展只在用户触发相关操作，或用户主动开启进入页面自动总结时处理页面内容。
